CHANGELOG Sentrifugo
===========================

RELEASE 2.0.1
-------------

Enhancements

- Removed rewrite module dependency during Sentrifugo Installation
- Updated PHP Mailer component to 5.2.10
- Enabled select and add new currency feature in Configuration Wizard
- Enabled few more widgets for Dashboard
- Changed mail template UI
- Enabled Mailing functionality and Activity Logs for Performance Appraisal module

Bug fixes

- Fixed issues in Performance Appraisal module         
- Fixed issues in Employee documents functionality
